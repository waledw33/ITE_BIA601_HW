Adjust genetic algorithm parameters including population size, generations, crossover rate, and mutation rate.

4. Analyze Results

Review selected features, fitness scores, and evolution visualizations. Export results for further analysis.

Algorithm Parameters

Parameter Range Default Description
Population Size 20-200 50 Number of candidate solutions in each generation. Larger populations explore more solutions but increase computation time.
Generations 50-500 100 Number of evolutionary cycles. More generations allow better convergence but increase runtime.
Crossover Rate 0.5-1.0 0.8 Probability of combining parent genes. Higher values increase exploration of gene combinations.
Mutation Rate 0.01-0.5 0.1 Probability of random gene changes. Higher values prevent premature convergence but may slow optimization.
Elitism Rate 0.05-0.2 0.1 Proportion of best individuals preserved each generation. Ensures top solutions are not lost.
Alpha (α) 0.0-1.0 0.9 Weight for accuracy in fitness function. Higher values prioritize accuracy over feature reduction.

Command-Line Interface

Execute experiments directly from the terminal using the CLI interface:

Basic Usage python cli_runner.py --dataset data.csv --target class_column

Advanced Configuration

python cli_runner.py \
    --dataset data.csv \
    --target outcome \
    --population 100 \
    --generations 200 \
    --crossover 0.8 \
    --mutation 0.1 \
    --model random_forest \
    --output results.json \
    --plot \
    --verbose

Python API

Integrate the framework into your Python code: 

from genetic_algorithm.ga_engine import GeneticAlgorithm
from sklearn.datasets import load_iris

# Load dataset
X, y = load_iris(return_X_y=True)

# Configure genetic algorithm
ga = GeneticAlgorithm(
    population_size=50,
    n_generations=100,
    crossover_rate=0.8,
    mutation_rate=0.1,
    elitism_rate=0.1,
    alpha=0.9,
    model_type='random_forest',
    verbose=True
)

# Execute optimization
best_solution = ga.fit(X, y)

# Get results
selected_features = ga.get_selected_features()
print(f"Selected features: {selected_features}")

# Visualize evolution
ga.plot_history()

Troubleshooting

Algorithm Not Converging

• Increase the number of generations
• Adjust mutation rate to prevent local optima
• Try different selection methods
• Increase population size for better exploration

Low Classification Accuracy

• Increase alpha (α) value to prioritize accuracy
• Try different ML models (RF, SVM, KNN)
• Ensure proper data preprocessing
• Check for class imbalance in target variable

Performance Issues

• Reduce population size for faster execution
• Decrease number of generations
• Use fast evaluation mode (single train-test split)
• Consider feature preprocessing to reduce dimensionality
